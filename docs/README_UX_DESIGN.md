## ðŸŽ¨ Design System

All UI built using **Seamstress Design System** (OpenGov's component library):

**Components Included:**
- Buttons (primary, secondary, tertiary)
- Forms (inputs, selects, checkboxes, radio)
- Cards, badges, alerts, modals
- Navigation (topbar, breadcrumb, sidebar)
- Tables and data displays
- State selector (unique colors for each state)

**Design Tokens:**
- Typography: H1-H6, body text (16px base)
- Colors: Primary brand, status colors, state-specific colors
- Spacing: 8px base unit (8, 16, 24, 32, 48px)
- Accessibility: WCAG AA compliant
